# Website source

Source code for my personal website. This iteration uses custom code based on
[Tornado](https://github.com/tornadoweb/tornado).

## Installing requirements

```
$ pip install -r requirements.txt
```

## Serving

```
$ python app.py serve
```

## Building

```
$ python app.py generate
```

or

```
$ nox -s build
```
